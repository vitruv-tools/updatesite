Vitruvius Updatesites

To install Vitruvius, you can add our nightly or release updatesites within Eclipse and start installing features from them:
  • Nightly: https://vitruv-tools.github.io/updatesite/nightly/aggregated
  • Release: https://vitruv-tools.github.io/updatesite/release

Within Eclipse, navigate to Help - Install New Software and click on the option.

In the new window, paste the updatesite link into Work with:, press enter, and select the plugins you want to install.
